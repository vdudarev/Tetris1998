### TETRIS

Это написанный мной когда-то в далеком 1998 году на первом курсе института Tetris (на Pascal под MS-DOS).

Для сборки исходников можно вооружиться Borland Turbo Pascal 7.0 (скомпилированная версия в папке Distrib - tetris.exe).

Запускать в современных реалиях надо через эмулятор типа https://www.dosbox.com/ - на удивление, все работает.

Ну и все в оригинале (осторожно, кодировка DOS-866)!
